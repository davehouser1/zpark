Welcome to Zpark's documentation!
=================================

.. toctree::
   :maxdepth: 2
   :caption: Contents:

Zpark is a bot that creates an integration between the `Cisco Spark`_
messaging platform and the real-time monitoring software `Zabbix`_.

This documentation covers the Python module internals and API. The user guide
which describes how to install and use the bot is at
`knightjoel.github.io/zpark`_.
