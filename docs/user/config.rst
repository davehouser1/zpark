Configuration
=============

Instructions
------------

Zpark ships with a default configuration that **must be modified before
use**. The default settings are stored in the :py:mod:`zpark.default_settings`
module complete with detailed documentation about each configuration setting.

Please **refer to the** `user guide`_ for detailed instructions on installing
and configuring this software.
